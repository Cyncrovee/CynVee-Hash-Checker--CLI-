## CynVee-Hash-Checker (CLI)

The CLI version of CynVee-Hash-Checker, written in Python.

You can find the GUI version here: https://github.com/Cyncrovee/CynVee-Hash-Checker

### Currently supports:
- SHA1
- SHA224
- SHA256
- SHA384
- SHA512
- SHA3_224
- SHA3_256
- SHA3_384
- SHA3_512
- MD5
- BLAKE2b
- BLAKE2s
